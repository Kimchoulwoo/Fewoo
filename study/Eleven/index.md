## 개발환경
- 언어 : Java
- DBMS : MySQL
- 운영체제 : Android
- 개발도구 : Android Studio

## 프로젝트 설명
- 목적 : 일상생활에 어려움을 겪는 청각장애인들을 돕기 위한 어플 설계 및 제작
- 인원 : 4명
- 기간 : 2016.09 ~ 2016.12 (3달)
- 기능 : 수화검색, 음성번역, 진동알람, SOS 사이렌, 복지시설 안내
- 상세설명 : 설계과목 수강 중 개발한 어플로 설계과정에 중점을 두고 개발해서 완성도가 많이 떨어짐. Google 검색 및 Youtube를 참고해 개발
